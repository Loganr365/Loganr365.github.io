Crowdsassins is a multiplayer game created in unity. Players must find each other in large crowded areas by cheating off each other's screens. The first player to find and knock out the other one wins. Be warned, if a player attacks an innocent they will be justly punished.

For this project, I worked heavily on the AI scripting. Each NPC randomly wanders around the area. This was done by having each one seek out an invisible game object, located, but not rendered in the world. This object would randomly change positions as time passed.
